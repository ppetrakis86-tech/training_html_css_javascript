#This is a practice project for HTML CSS and JAVASCRIPT coding

Here are your 40 exercises, grouped by topic, easy to hard:

---

**HTML (10 exercises)**

Folder /Html practice

1. Create a basic HTML page with a title, one heading, and one paragraph.
2. Add an image and a link to your page.
3. Build an unordered list and an ordered list.
4. Create a table with 3 columns and 4 rows.
5. Build a simple form with a text input, email input, and a submit button.
6. Use semantic tags — header, nav, main, footer — to structure a page.
7. Create a page with two sections and an aside column.
8. Embed a YouTube video using an iframe.
9. Build a navigation menu with links using a nav tag and ul/li.
10. Create a full page layout: header with logo, nav, main content area with an article and sidebar, and a footer.

**CSS (10 exercises)**

1. Change the background color and font color of a page.
2. Style a heading with a custom font size, weight, and color.
3. Add padding and margin to a div to control spacing.
4. Create a box with a border, rounded corners, and a box shadow.
5. Style a button with a hover effect that changes color smoothly using transition.
6. Center a div horizontally and vertically using Flexbox.
7. Build a row of three equal boxes side by side using Flexbox.
8. Create a card component with an image, title, and text, styled nicely.
9. Make a simple responsive layout that stacks vertically on small screens using Flexbox.
10. Build a full page layout with header, sidebar, main content, and footer using only Flexbox.

1. Write a function that shows an alert when a button is clicked.
2. Change the text of a paragraph when a button is clicked.
3. Toggle the visibility of a div on button click.
4. Create a counter that increases and decreases with two buttons.
5. Change the background color of the page randomly on button click.
6. Build a simple to-do list where you can add items by typing and pressing a button.
7. Create a form that validates if the email field is empty before submitting.
8. Fetch and display a joke from a public API on button click.
